# Adaptive Multi-Cell Battery Intelligence Engine

## Overview
This project implements a Battery Intelligence Engine using Arduino Uno and Wokwi simulation for monitoring a 4-cell lithium battery pack in real time.

## Features
- Individual cell voltage monitoring
- Pack voltage calculation
- Average voltage calculation
- Strongest and weakest cell identification
- Battery imbalance percentage calculation
- State of Charge (SOC) estimation
- Cell balancing recommendation
- Temperature monitoring
- Battery health classification

## Battery States
- Healthy
- Minor Imbalance
- Critical Imbalance
- Pack Failure

## Components Used
- Arduino Uno
- 4 Potentiometers
- Wokwi Simulator
